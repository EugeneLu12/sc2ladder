About

sc2Ladder is a web app that displays the in-game ladders of StarCraft II and updates them in real time. It also has the unique feature of revealing an account's Battle Net ID, revealing the identity of some barcode accounts.

The source code can be found here.

All inquiries can be sent to YenFu#4967 via private message on Discord.

Special thanks to tso, woopr, and Buster for their contributions to this project.

API Documentation

The API Endpoints available all return a JSON object and have a quota of 10000 requests/hour. Continuing to make requests after this quota is reached results in a 429 Error.

All endpoints limit requests to 25 results returned by default but the limit can be modified to return up to 200 results.

GET - Player

Returns players whose character name or battletag matches the query parameter.

Request URL Format: /api/player?query={query}&limit={limit}

Parameter Data Type Value Description
query string The player's character name or battletag
limit int Number of results returned
